US Interconnection Atlas: the grid interconnection queue on a D3 map

An interactive choropleth of US grid interconnection. Pick a metric and the 50 states + DC recolor live; interconnection-queue projects plot as points sized by capacity, with ISO/RTO territories (CAISO, ERCOT, MISO, SPP, PJM, NYISO, ISO-NE) outlined on top. Click a state and the detail panel rescopes to it.

Real mode draws from EIA-860M (planned generators with coordinates and nameplate capacity) and HIFLD control-area footprints, bundled as a static snapshot: no backend, everything runs in the browser. Synthetic mode is a seeded illustrative dataset, not actual filings.

Open source on GitHub, by John Carmack. Built with D3 and React.
